# 01 Read Topic Builder Outputs

Read the available `paper-topic-builder` outputs and build a source inventory.

Required actions:

1. Identify which expected files are available.
2. Extract selected topic candidates, theory bases, variables, contexts, methods, reviewer risks, and evidence notes.
3. Preserve source file names and evidence locations.
4. Mark unavailable files as missing.
5. Do not infer facts that are not present in the source outputs.

Output a concise source summary and a structured source index for later steps.
