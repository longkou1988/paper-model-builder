# 02 Select Topic And Model Focus

Use the source inventory and optional user notes to identify the model focus.

Required actions:

1. Confirm the selected topic.
2. Identify the core management problem.
3. Identify the research question.
4. Identify the theoretical foundation.
5. Identify the empirical context and research object.
6. Decide whether the model should be direct, mediated, moderated, moderated mediation, chain mediation, multi-level, configuration, or process-oriented.
7. If topic selection is ambiguous, ask the user to choose before generating the final model.

Do not create a model before the selected topic is clear.
