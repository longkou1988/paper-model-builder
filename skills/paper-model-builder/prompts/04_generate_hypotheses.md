# 04 Generate Hypotheses

Generate hypothesis paths from the model specification.

For each hypothesis, include:

1. Hypothesis ID.
2. Path type.
3. Source variable.
4. Target variable.
5. Mediator or moderator when applicable.
6. Expected direction.
7. Chinese hypothesis text.
8. English hypothesis text.
9. Theoretical rationale.
10. Evidence source.
11. Confidence.
12. Reviewer risk.

Do not write unsupported hypotheses. If a path is only a researcher-proposed extension, label it clearly.
