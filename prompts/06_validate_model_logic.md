# 06 Validate Model Logic

Review the model from an SSCI management journal reviewer perspective.

Check:

1. Whether the research question is clear.
2. Whether the theoretical contribution is explicit.
3. Whether variable roles are coherent.
4. Whether mediators explain mechanisms.
5. Whether moderators represent real boundary conditions.
6. Whether the model is overcomplicated.
7. Whether the method matches the model.
8. Whether data can realistically be collected.
9. Whether common method bias, endogeneity, or construct overlap risks exist.
10. Whether the model looks like a variable patchwork.

Provide revision suggestions, not only criticism.
